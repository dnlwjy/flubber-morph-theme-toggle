# flubber-morph-theme-toggle

A React theme toggle component that morphs between a crescent moon and sun using **Flubber** path interpolation and **Framer Motion** animation.

---

## How to Build SVG Path Morphing with Flubber

This guide walks you through creating a smooth SVG shape morphing animation from scratch — the same technique used in this component to morph a moon into a sun circle.

### Step 1 — Install dependencies

```bash
npm install flubber framer-motion
```

- **flubber** — interpolates between two SVG path strings
- **framer-motion** — drives the animation and motion values

---

### Step 2 — Get your two SVG paths

You need two SVG `path` `d` strings that share the same coordinate space (same viewBox).

```ts
// A crescent moon shape
const MOON_PATH = "M28.752 22.002A9.72 9.72 0 0 1 25 22.75c-5.385 0-9.75-4.365-9.75-9.75 ..."

// A circle — generated by flubber from the moon path
import { toCircle } from "flubber"
const CIRCLE_PATH = toCircle(MOON_PATH, 19, 19, 5)(1)
//                                       cx  cy  r
```

`toCircle(sourcePath, cx, cy, radius)` returns an interpolator. Calling it with `(1)` gives you the final circle shape as a path string.

> Tip: Both paths must live in the same SVG `viewBox` coordinate space so they overlap correctly during morphing.

---

### Step 3 — Create the morph interpolators

```ts
import { interpolate } from "flubber"

const moonToCircle = interpolate(MOON_PATH, CIRCLE_PATH, { maxSegmentLength: 1.5 })
const circleToMoon = interpolate(CIRCLE_PATH, MOON_PATH, { maxSegmentLength: 1.5 })
```

`maxSegmentLength` controls how many intermediate points flubber adds. Lower = smoother morph, higher = better performance.

Wrap these in `useMemo` so they are only computed once:

```ts
const [moonToCircle, circleToMoon] = useMemo(() => [
  interpolate(MOON_PATH, CIRCLE_PATH, { maxSegmentLength: 1.5 }),
  interpolate(CIRCLE_PATH, MOON_PATH, { maxSegmentLength: 1.5 }),
], [])
```

---

### Step 4 — Drive the animation with a motion value

```ts
import { useMotionValue, animate } from "framer-motion"

const progress = useMotionValue(0) // 0 = start shape, 1 = end shape
```

Subscribe to progress changes and update the rendered path string:

```ts
const [pathD, setPathD] = useState(MOON_PATH)

useEffect(() => {
  return progress.on("change", (t) => {
    setPathD(goingToCircle ? moonToCircle(t) : circleToMoon(1 - t))
  })
}, [progress, goingToCircle])
```

Trigger the animation on toggle:

```ts
animate(progress, isDark ? 1 : 0, {
  duration: 0.5,
  ease: [0.4, 0, 0.2, 1],
})
```

---

### Step 5 — Render with `motion.path`

Use `motion.path` (or `m.path` for lazy loading) and pass the interpolated `d` value:

```tsx
import { motion } from "framer-motion"

<svg viewBox="0 0 38 38" fill="none">
  <motion.path
    d={pathD}
    fill="none"
    stroke="currentColor"
    strokeWidth={1}
    style={{ originX: 0.5, originY: 0.5 }}
  />
</svg>
```

> `originX: 0.5, originY: 0.5` ensures any scale/rotate transforms are centered on the shape.

---

### Step 6 — Put it all together

```tsx
'use client'

import { useState, useEffect, useMemo } from "react"
import { motion, useMotionValue, animate } from "framer-motion"
import { interpolate, toCircle } from "flubber"

const MOON_PATH = "M28.752 22.002..." // your moon path
const CIRCLE_PATH = toCircle(MOON_PATH, 19, 19, 5)(1)

export function MorphToggle() {
  const [isDark, setIsDark] = useState(false)
  const progress = useMotionValue(0)
  const [pathD, setPathD] = useState(MOON_PATH)

  const [moonToCircle, circleToMoon] = useMemo(() => [
    interpolate(MOON_PATH, CIRCLE_PATH, { maxSegmentLength: 1.5 }),
    interpolate(CIRCLE_PATH, MOON_PATH, { maxSegmentLength: 1.5 }),
  ], [])

  useEffect(() => {
    return progress.on("change", (t) => {
      setPathD(isDark ? moonToCircle(t) : circleToMoon(1 - t))
    })
  }, [progress, isDark])

  const toggle = () => {
    const next = !isDark
    setIsDark(next)
    animate(progress, next ? 1 : 0, { duration: 0.5, ease: [0.4, 0, 0.2, 1] })
  }

  return (
    <button onClick={toggle} style={{ width: 80, height: 80 }}>
      <svg width="100%" height="100%" viewBox="0 0 38 38" fill="none">
        <motion.path
          d={pathD}
          stroke="currentColor"
          strokeWidth={1}
          fill="none"
          style={{ originX: 0.5, originY: 0.5 }}
        />
      </svg>
    </button>
  )
}
```

---

### Key concepts recap

| Concept | Role |
|---|---|
| `flubber.interpolate(a, b)` | Creates a function `t => pathString` that morphs from `a` to `b` |
| `flubber.toCircle(path, cx, cy, r)` | Generates a circle path compatible with the source path's point count |
| `useMotionValue` | Holds the animation progress `0→1` without causing re-renders |
| `progress.on("change")` | Subscribes to every frame of the animation to update `pathD` |
| `animate(progress, target)` | Smoothly drives the progress value with easing |
| `motion.path` | Renders the SVG path and supports framer-motion transitions |

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `size` | `number` | `80` | Width and height in px |
| `strokeWidth` | `number` | `1` | Stroke thickness |
